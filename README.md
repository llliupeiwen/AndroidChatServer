# AndroidServer
仿微信服务器demo

ip地址和数据库连接按情况配置

注意：webapp/WEB-INF/lib下有两个包需要build

已实现：
  1、查找数据库中的聊天记录
  2、查找数据库中的好友关系
  3、账号登录的查询
  4、账号注册的修改
  5、Socket实现1对1聊天服务
